White Rabbit v3.7 — заказ на карточке столика и создание заказа официантом

Что исправлено
1. Администратор: в разделе «Столики» нажмите красную карточку занятого столика.
   В нижней таблице «Заказ выбранного столика» появится активный заказ:
   номер, статус, источник, клиент, официант, время создания и счёт.
2. Официант: после открытия смены и загрузки назначенных столиков кнопка
   «Создать заказ» становится активной. Выберите карточку столика, укажите
   гостей и создайте заказ.

SQL запускать не нужно, если база установлена единым скриптом v4.0.

Перед запуском:
- Закройте приложение.
- Откройте WhiteRabbitRestaurant.sln в Visual Studio.
- Выполните Build -> Rebuild Solution.
