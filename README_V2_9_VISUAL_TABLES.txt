White Rabbit v2.9 — визуальные столики, брони и фильтры смен

Что изменено
1. У администратора вкладка «Столики» теперь показывает интерактивные карточки столиков вместо таблицы.
2. В рабочем месте официанта назначенные столики отображаются карточками. Нажатие на карточку выбирает столик для нового заказа.
3. У администратора на вкладке «Бронирование» добавлена схема столиков по выбранной дате.
   После нажатия на карточку столика ниже выводятся все активные брони этого столика за выбранный день.
4. В «Смены и столики» добавлены фильтры по дате, официанту, статусу и типу смены.

Как установить обновление
1. Откройте SQL Server Management Studio.
2. Выберите базу WhiteRabbitRestaurant.
3. Выполните SQL_Update_v2_9_Visual_Table_Maps_And_Shift_Filters.sql.
4. Откройте проект в Visual Studio.
5. Выполните Restore NuGet Packages, Clean Solution, Rebuild Solution и F5.

Легенда карточек
- зелёная — столик свободен;
- жёлтая — столик забронирован или имеет брони на выбранный день;
- красная — столик занят активным заказом;
- синяя — столик назначен текущему официанту;
- бордовая — выбранный столик.
