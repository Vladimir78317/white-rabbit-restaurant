White Rabbit v3.6 — исправление пустого списка столиков

Что исправлено:
- при отсутствии свободных столиков приложение больше не пытается выбрать несуществующий первый пункт списка;
- безопасно обновляются списки столиков и официантов;
- убрана ошибка WinForms: "value ('0') must be less than '0'".

База данных не меняется. SQL-скрипты запускать не требуется.

Как применить:
1. Закройте White Rabbit.
2. Откройте файл WhiteRabbitRestaurant.sln из этого архива в Visual Studio.
3. Удалите папки bin и obj, если они остались в папке проекта.
4. Нажмите Build -> Rebuild Solution, затем запустите приложение.
